Popsicle Stick Bridge Competition - 2021
Semi-automated bridge stress tester.

Gitlab Link to source code.
My team built a combination linear actuator, motor driver, HX711 load cell and Raspberry Pi 3+ on a table to break popsicle stick bridges and record the strength at break point. HX711 load cell needed manual calibration for each individual load cell. Reading the load cell while actuating the linear actuator and displaying the weight required making a background process in a seperate thread so that the UI would not break.
500+ Popsicle stick bridges were built and designed by Fort Bend ISD elementary school students within guidelines set by our team. Previous manual bridge testing took 2 days, and over 100 volunteers for over 200 bridges. Two of these semi-automated tables destroyed and recorded over 500 bridges with less than 20 volunteers in 6 hours.

The script was written in Python 3 with a PyQT5 UI frontend and SQLite backend to store scores of bridges tested. Testing requires manually starting with UI, script then captures weight on bridge, displays weight and automatically stops at breakpoint. The final weight is displayed and can be manually stored with the team name in seperate part of UI.
